## Limiti
- **Speed**
Real-time o design-time?
- **Reliability**
Catastrophic failures break gameplay
- **Controllability**
Allow specification of constraints and goals
- **Diversity**
Content looks like variations on a theme
- **Creativity**
Content looks "computer-generated
